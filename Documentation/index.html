Installation

The zip file contains all js files integrated with Lanrdick, however, you need to perform the following steps to get node_modules, etc.

Minimum Node Version Required

 10

Open terminal or command prompt from the root directory of the project and run below command.

 npm install

After Completion of installation Run following command to run the project locally. It will run automatically on http://localhost:3000 URL in your default browser it a couple of minutes to run.

npm start

For production build run the below command.

npm build

Theme settings

How to change template mode?
You can simply change this template's mode ( Dark, Light, RTL, Dark RTL ) by changing single line in the file /src/Apps.scss file. We have commented all versions css which is very easy to change its mode from Dark to Light / RTL / Dark RTL etc.
We suggest you to write your custom scss in custom.scss file instead of updating other scss files.

How to change menu items?
You can simply add or remove menu items from the src/components/Layout/Topbar.js file.
